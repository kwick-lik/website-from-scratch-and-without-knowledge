Шаг 1: Установка и настройка инструментов

Для создания сайта тебе понадобятся бесплатные инструменты, которые установит любой начинающий разработчик.

📥 Что нужно установить:

Шаг 2: Создание базовой HTML-структуры

Каждый сайт начинается с HTML-файла. Это каркас твоего будущего сайта.

📝 Создай файл index.html с таким кодом:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Мой первый сайт</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это мой первый сайт.</p>
</body>
</html>

💡 Что означает этот код:

  • <!DOCTYPE html> - говорит браузеру, что это HTML5
  • <html lang="ru"> - корневой элемент с указанием языка
  • <head> - служебная информация (не видна на странице)
  • <body> - видимое содержимое страницы

Шаг 3: Добавление стилей с помощью CSS

CSS делает твой сайт красивым. Создай файл style.css и подключи его к HTML.

🎨 Создай файл style.css:

/* Основные стили для тела страницы */
body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
    margin: 0;
    padding: 20px;
    background-color: #f4f4f4;
}

/* Стили для заголовка */
h1 {
    color: #333;
    text-align: center;
}

/* Стили для параграфа */
p {
    color: #666;
    font-size: 18px;
}

🔗 Подключи CSS к HTML:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Мой первый сайт</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это мой первый сайт.</p>
</body>
</html>

Шаг 4: Добавление интерактивности с JavaScript

JavaScript делает сайт "живым". Создай файл script.js для добавления интерактивных элементов.

⚡ Создай файл script.js:

// Этот код выполнится при загрузке страницы
console.log("Сайт загружен!");

// Добавим реакцию на клик по заголовку
document.querySelector('h1').addEventListener('click', function() {
    this.style.color = this.style.color === 'red' ? '#333' : 'red';
});

// Функция для показа сообщения
function showMessage() {
    alert('Привет от JavaScript!');
}

🔗 Подключи JavaScript к HTML:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Мой первый сайт</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это мой первый сайт.</p>
    <button onclick="showMessage()">Нажми меня</button>
    
    <script src="script.js"></script>
</body>
</html>
🎯 Нажми меня! Увидишь, как работает JavaScript

Шаг 5: Создание адаптивного дизайна

Современные сайты должны хорошо выглядеть на всех устройствах - от компьютеров до телефонов.

📱 Добавь в style.css медиа-запросы:

/* Базовые стили для больших экранов */
.container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 20px;
}

/* Для планшетов */
@media (max-width: 768px) {
    .container {
        padding: 15px;
    }
    h1 {
        font-size: 24px;
    }
}

/* Для телефонов */
@media (max-width: 480px) {
    .container {
        padding: 10px;
    }
    h1 {
        font-size: 20px;
    }
    p {
        font-size: 16px;
    }
}

🔄 Обнови HTML для адаптивности:

<body>
    <div class="container">
        <h1>Привет, мир!</h1>
        <p>Этот сайт адаптируется под любой размер экрана.</p>
        <button onclick="showMessage()">Нажми меня</button>
    </div>
    <script src="script.js"></script>
</body>

Шаг 6: Запуск и тестирование сайта

После написания кода нужно проверить, как сайт работает в браузере.

🚀 Как запустить сайт:

  • Открой папку с проектом в VS Code
  • Установи расширение Live Server
  • Кликни правой кнопкой на index.html
  • Выбери "Open with Live Server"
  • Сайт откроется в браузере на http://localhost:5500

🧪 Что тестировать:

  • Открывается ли сайт в разных браузерах
  • Корректно ли отображается на телефоне (F12 → Toggle device toolbar)
  • Работают ли все кнопки и ссылки
  • Не появляются ли ошибки в консоли (F12 → Console)
🎯 Нажми меня! Проверим твои знания

Шаг 7: Публикация сайта в интернете

Готовый сайт можно разместить в интернете бесплатно.

🌐 Бесплатные хостинги для начинающих:

  • GitHub Pages - для проектов на GitHub
  • Netlify - простой деплой перетаскиванием
  • Vercel - для современных веб-проектов
  • GitLab Pages - альтернатива GitHub

📦 Как опубликовать на GitHub Pages:

  1. Создай аккаунт на GitHub.com
  2. Создай новый репозиторий
  3. Загрузи файлы проекта (index.html, style.css, script.js)
  4. В настройках репозитория найди "Pages"
  5. Выбери ветку main и сохрани
  6. Через пару минут твой сайт будет доступен по адресу: username.github.io/repository-name
